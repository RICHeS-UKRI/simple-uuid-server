# Documentation

This directory contains documentation for the **Simple UUID Server**.

New users should typically start with **INSTALL.md**, followed by **USAGE.md**.

## Contents

- **[INSTALL.md](INSTALL.md)**  
  Instructions for installing and running the service as a systemd-managed
  Linux service, including permissions, configuration, and reverse proxy setup.

- **[USAGE.md](USAGE.md)**  
  Practical guidance on using the running service, including available
  endpoints, authentication, request formats, and example API calls.

## Notes

Additional documentation may be added over time as usage patterns and
integration requirements evolve.

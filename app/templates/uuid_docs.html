UUID Registry Service

This service returns a stable UUID for a given namespace, entity_type, and metadata.

Health

GET /health

Mint / Retrieve UUID

POST /uuid

Header: X-API-Key: <your key>

{
  "namespace": "HSDS",
  "entity_type": "PaintSample",
  "metadata": {
    "sample_code": "NG1234-L1",
    "painting": "NG1234",
    "location": "centre-left"
  }
}

Responses

  • 200 JSON with UUID and metadata
  • 400 Invalid request
  • 401 Invalid API key
